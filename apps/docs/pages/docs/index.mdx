# 🧞 Next Admin

###### `next-admin` is a library built on top of [Prisma](https://www.prisma.io/) and [Next.js](https://nextjs.org/) that allows you to easily manage and visualize your Prisma database in a nice GUI.

Get started by following the [installation guide](/docs/getting-started)

## ✨ Features

This library is still under development. The following features are available:

- 👀 Visualize, search and filter your data quickly
- 💅 Customizable admin dashboard
- 💽 Database relationships management
- 👩🏻‍💻 User management (CRUD operations)
- 🎨 Dashboard widgets and customizable panels
- ⚛️ Integration with Prisma ORM
- 👔 Customizable list and form
- ⚙️ Supports App Router and Page Router

If you want to request a feature, please open an issue [here](https://github.com/sseuniverse/next-admin/issues/new/choose)
